## Usage

```jsx copy
import {
  Accordion,
  AccordionItem,
  AccordionButton,
  AccordionContent,
} from "@components/Accordion/Accordion";
import {cn} from "@lib/utils/cn"

function AccordionDemo() {
  return (
    <Accordion variant="rounded" border={true}>
      <AccordionItem index={0}>
        <AccordionButton>Section 1</AccordionButton>
        <AccordionContent>Content for section 1</AccordionContent>
      </AccordionItem>
      <AccordionItem index={1}>
        <AccordionButton>Section 2</AccordionButton>
        <AccordionContent>Content for section 2</AccordionContent>
      </AccordionItem>
    </Accordion>
  );
}
```

## Components

### Accordion

The main wrapper component for the accordion.

#### Props

- `children`: ReactNode - The accordion items.
- `className`: ClassNameValue (optional) - Additional CSS classes.
- `variant`: "default" | "rounded" | "rectangle" (optional) - The visual style of the accordion.
- `border`: boolean (optional) - Whether to show a border around the accordion.

### AccordionItem

Represents a single item in the accordion.

#### Props

- `index`: number - The index of the item in the accordion.
- `children`: ReactNode - The content of the item (should include AccordionButton and AccordionContent).
- `className`: ClassNameValue (optional) - Additional CSS classes.

### AccordionButton

The clickable header of an accordion item.

#### Props

- `isActive`: boolean (optional) - Whether the item is currently expanded.
- `onClick`: () => void (optional) - Function to call when the button is clicked.
- `className`: ClassNameValue (optional) - Additional CSS classes.
- `children`: ReactNode - The content of the button.

### AccordionContent

The expandable content of an accordion item.

#### Props

- `className`: string (optional) - Additional CSS classes.
- `children`: ReactNode - The content to display when the item is expanded.
