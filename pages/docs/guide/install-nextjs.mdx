### Install Next.js

Create a Next.js with Create Next App and select the options described below,

### Create a new project

<CodeCopy
  code={`npx create-next-app@latest my-nextjs-app`}
  language="markdown"
/>

Would you like to use \`src/\` directory? No / Yes [Select No]
